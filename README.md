# Lumina Reader

A modern, dark-themed book reader designed for English documents.

## Features

- **Format Support**: Open and read EPUB, PDF, and TXT files.
- **Dark Theme**: Easy on the eyes for night-time reading.
- **Text-to-Speech (TTS)**: Integrated realistic English narration using the Web Speech API.
- **Customization**: Adjust font size and line height to suit your preference.
- **Navigation**: Simple pagination for EPUB and PDF files.

## How to use

1. Click the "Folder" icon in the top left to upload a book file.
2. Use the "Settings" icon to customize the text layout.
3. Use the bottom toolbar to control narration:
   - Select a voice from the dropdown (High-quality English US voices recommended).
   - Adjust playback speed.
   - Play, Pause, or Stop reading.

## Technical Notes

- Built with **Epub.js** for EPUB rendering.
- Built with **PDF.js** for PDF rendering.
- Powered by the **Web Speech API** for natural-sounding English narration.
- Fully responsive and optimized for mobile devices.
